# Portfolio · Ciencia de Datos e Inteligencia Artificial

Portfolio web desarrollado como entrega del curso de Ciencia de Datos e IA.
Presenta tres proyectos realizados a lo largo del curso.

## 🚀 Ver el portfolio

Abre `index.html` en cualquier navegador, o publícalo con GitHub Pages.

## 📁 Estructura

```
portfolio/
├── index.html       # Página principal del portfolio
└── README.md        # Este archivo
```

## 🛠️ Cómo publicar en GitHub Pages

1. Sube el repositorio a GitHub
2. Ve a **Settings → Pages**
3. En *Source*, selecciona la rama `main` y carpeta `/ (root)`
4. Guarda y espera unos segundos — tu portfolio estará en `https://tu-usuario.github.io/nombre-repo`

## ✏️ Cómo personalizar

Edita `index.html` y reemplaza:

| Placeholder | Por |
|---|---|
| `Tu Nombre` | Tu nombre real |
| `@tu-usuario` | Tu usuario de GitHub |
| Proyecto 01, 02, 03 | Títulos y descripciones de tus proyectos reales |
| `href="#"` en cada tarjeta | URL de tu notebook o repositorio de cada proyecto |

## 📦 Tecnologías

- HTML5 / CSS3 / JavaScript vanilla
- Google Fonts (DM Serif Display + DM Sans)
- Sin dependencias ni frameworks

## 📄 Proyectos incluidos

1. **Análisis de Ventas en E-commerce** — EDA y visualización
2. **Predicción de Abandono de Clientes** — Clasificación con ML
3. **Clasificación de Imágenes con CNN** — Deep Learning

> Reemplaza estos proyectos con los tuyos propios.
